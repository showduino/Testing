What is Showduino?

A control desk + execution brain

Showduino mirrors a professional production pipeline: the UI ESP32-S3 deck (JC8048W550C + GT911) handles LVGL visuals, while the separate Brain ESP32-S3 fires relays, LEDs, FX, DMX and MP3 playback. The two boards exchange structured JSON over UART or Wi-Fi.

UI Deck

Touchscreen console running LVGL, Montserrat UI, session-aware timelines.

  • Manual / Auto control modes
  • Live timeline scrubbing & cue stack
  • Relays, LEDs, MP3 widgets mirrored from Brain
  • Status bar, toast alerts, Wi-Fi + serial handoff
{"cmd":"relay","id":3,"state":1} {"cmd":"led","line":1,"pixel":42} {"status":"ok","mode":"auto"}

Showduino Brain

ESP32-S3 control hub executing every physical action safely.

  • Relay, DMX, SX1509 expansion
  • Pixel matrices, WS2812B & power domains
  • DFPlayer dual audio decks + SD logging
  • PrizmLink prop auto-detection via I²C
